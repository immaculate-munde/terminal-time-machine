# Terminal Time Machine ⏳

An interactive Git timeline in your terminal with AI-generated commit summaries.

## Installation

### Local Development
1. Clone this repo.
2. Run `npm install`.
3. Create a `.env` file with `GITHUB_TOKEN=your-token`.
4. Run `npm start`.

### Global Installation
You can link this locally to use it anywhere:
```bash
npm link
# Then you can run:
time-machine
```

## Requirements
- Node.js v20+
- Git
- A [GitHub Personal Access Token](https://github.com/settings/tokens) or Gemini API Key.
